Introduction

Invoza is a multi purpose Bootstrap 4 landing page template built with React js.This responsive template comes with 6 Demo variations which can be used to build your product, service, marketing, app and SaaS landing pages. It’s fully responsive and built with bootstrap v4.5.0. You will be easily able to customize it for your needs.

Please feel free to get back to me in case if you are having any question or feedback.

App.js Structure

import React, { Component } from "react";
import routes from "./routes";
import {
	withRouter,
	Route,
	Switch,
	BrowserRouter as Router
} from "react-router-dom";

// Import Scss
import "./theme.scss";

//Import Icon Css
import "./assets/css/materialdesignicons.min.css";

class App extends Component {
	constructor(props){
		super(props);
		this.state = {};
	}
	render() {
		return (
			<React.Fragment>
				<Router>
					<Switch>
						{routes.map((route, idx) => (
						<Route path={route.path} component={route.component} key={idx} />
						))}
					</Switch>
				</Router>
			</React.Fragment>
		);
	}
}
	
default withRouter(App);
